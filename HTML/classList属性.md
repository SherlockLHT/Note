- H5新增，JS通过之获取DOM节点的class；

#### Properties

| Property | Description |
| -------- | ----------- |
| length   | 数量        |

#### Method

| Method                     | Description     |
| -------------------------- | --------------- |
| add(class 1, class 2, ...) | 新增一个/多个类 |
| remove(class 1, class 2, ...) | 删除一个/多个类 |
| contain(class) | 是否包含 |
| item(index) | 返回类名 |
| toggle(class 1, true\|false) | 存在则删除，不存在则添加<br>true：强制添加<br>false：强制删除 |
